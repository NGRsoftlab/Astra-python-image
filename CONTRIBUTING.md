# Contributing

Пожалуйста, сначала публикуйте все вопросы и проблемы в `Issues`

## Contribution to the project

1. Способ внести свой вклад:
   1. Форкайте репозиторий
   2. Создавайте feature-ветку (`git checkout -b feature/AmazingFeature`)
   3. Коммитьте изменения (`git commit -m 'Add some AmazingFeature'`)
   4. Пушите в ветку (`git push origin feature/AmazingFeature`)
   5. Открывайте Pull Request
2. Если Вы хотите внести свой вклад в проект, то можете начать с прочтения [документации](README.md)
